Central Limit Theorem:CLT (중심극한정리)

Concept

임의의 표본들을 추출한 모집단의 확률분포에 상관없이 표본평균의 분포는 표본의 크기가 크면 클수록 정규분포에 분포수렴하게 된다. 중심극한정리는 통계학에서 중요한 정리 중 하나이므로 반드시 기억할 필요가 있다.

par(mfrow = c(1, 3))
bar_x <- bar_y <- bar_z <- numeric(500)
for (j in c(10, 100, 1000, 10000)) {
    for (i in 1:500) {
        x <- runif(j, 0, 1)
        bar_x[i] <- mean(x)
        y <- rexp(j, 1/3)
        bar_y[i] <- mean(y)
        z <- rchisq(j, 1)
        bar_z[i] <- mean(z)
    }
    hist(bar_x, prob = T, main = c("Uniform n=", j))
    curve(dnorm(x, 1/2, 1/sqrt(12 * j)), add = T, col = "red", lwd = 2)
    hist(bar_y, prob = T, main = c("Exponential n=", j))
    curve(dnorm(x, 3, 3/sqrt(j)), add = T, col = "blue", lwd = 2)
    hist(bar_z, prob = T, main = c("Chi-square n=", j))
    curve(dnorm(x, 1, sqrt(2/j)), add = T, col = "green", lwd = 2)
}
