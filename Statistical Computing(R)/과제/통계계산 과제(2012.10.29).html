명령문 for를 이용하여 각 변수가 factor인지를 살펴 그 여부를 출력하라.

result <- 0
for (i in 1:4) {
    result[i] <- is.factor(cabbages[, i])
}
print(result)  ## 여기서 TRUE는 1로, FALSE는 0으로 출력된다.

## [1] 1 1 0 0

명령문 while을 이용하여 VitC 값을 차례대로 더하여 500을 초과하면 몇 개의 관측치를 더하여 어떤 값을 얻었는지를 출력하여라.

sum <- 0
n <- 0
while (sum <= 500) {
    n <- n + 1
    sum <- sum + VitC[n]
}
print(sum)

## [1] 503

print(n)

## [1] 10
