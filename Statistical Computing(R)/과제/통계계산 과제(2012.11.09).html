Question

몬테카를로 적분을 이용한 적분함수를 아래와 같이 작성하여라.

- 함수형식 : monte.integral(a,b,c,n)

여기서 a, b, c는 의 parameter, n은 난수 개수

- 두 근이 존재하는 2차원의 convex(볼록) 함수에 대하여 함수 값이 0보다 큰 부분에 대한 적분을 구하는 함수 (근이 없거나 convex가 아니면 경고 후 정지)

- 우선 두 근이 존재하는지를 확인 한 후 구간 안에 convex 함수 값이 가장 큰 값을 높이로 하고 두 근의 구간을 밑변으로 하는 sample space를 확보한 뒤 구간 안에 0보다 크고 함수 값보다 작은 함수가 나올 확률을 구하여 넓이를 구하는 함수

먼저 우리가 얻고자 하는 함수 monte.integral(a,b,c,n) 에 대한 R code 를 작성하여보자.

library("UsingR")

## Loading required package: MASS

library("knitr")
monte.integral <- function(a, b, c, n) {
    t1 <- (-b - sqrt(b^2 - 4 * a * c))/(2 * a)
    t2 <- (-b + sqrt(b^2 - 4 * a * c))/(2 * a)
    t3 <- c - ((b^2)/(4 * a))
    t4 <- t1 - t2
    x <- runif(n, t2, t1)
    y <- runif(n, 0, t3)
    f <- y <= (a * (x^2) + b * x + c)
    prob <- mean(f)
    answer <- prob * (t3 * t4)
    return(answer)
}

여기에서 a,b,c는 임의의 2차 다항함수 y=ax²+bx+c의 3가지 모수이고 n은 발생시킬 난수의 개수를 의미한다.

t1과 t2는 방정식 ax²+bx+c=0 가 2개의 서로 다른 실근을 가질 경우의 해당하는 근을 나타낸 것이고 t3은 위의 2차함수가 위로 볼록(convex)하다고 가정할 때의 최댓값을 나타낸다.

또한 t4는 두 실근의 차이고(여기에서는 a가 음수인 경우만 수치가 출력되는 함수를 만들 것이므로 t1이 t2보다 큰 값임을 알 수 있다.) x와 y는 위 문제에서 제시한 sample space에서 균일하게 발생시킨 난수이다.

원하는 값을 구하기 위해 f에서 해당 2차함수보다 아래인 면적을 나타냈고, 몬테카를로 적분에 의해 생성되는 근사적인 확률 prob를 만들었으며, answer는 우리가 얻고자 하는 값이다. 다시 말하면 확률에 sample space의 면적을 곱하게 되면 우리가 원하는 근사적분값이 나오게 된다.

그러면 먼저 함수 y=-x²+4x-3 에 대해서 해당하는 면적의 정확한 값을 구해보자.

f1 <- function(x) -x^2 + 4 * x - 3
integrate(f1, 1, 3)

## 1.333 with absolute error < 1.5e-14
