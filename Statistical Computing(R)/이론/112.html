Package인 UsingR을 loading하여 Data frame인 kid.weights를
사용하여 아래 물음 및 지시에 해당하는 R 프로그램을 작성하시오.

library(UsingR)

## Loading required package: MASS

data(kid.weights)
attach(kid.weights)
  1. x축은 height, y축은 weight로 이루어진 plot을 그리시오. 또한 height를 독립변수로 하는 단순회귀분석 회귀선을 그리시오.
plot(height, weight, pch = as.character(gender))
reg <- lm(weight ~ height)
abline(reg)

  1. gender(F,M)별로 height의 평균을 구하고, 막대그래프를 그리시오. (단, x축이름은 gender로, y축이름은 height mean으로 지정하고 y축눈금은 0에서 40까지 나타나도록 하시오.)
man <- mean(height[gender == "M"])
female <- mean(height[gender == "F"])
mean <- cbind(man, female)
barplot(mean, xlab = "gender", ylab = "height mean", ylim = c(0, 40))

  1. 명령문 while을 이용하여 변수 gender의 값이 M인 것들의 weight값만을 차례대로 더하여 1000을 초과하면 몇 개의 관측치를 더하여 어떤 값을 얻었는지를 출력하시오.
sum.weight <- 0
n <- 0
while (sum.weight <= 1000) {
    n <- n + 1
    sum.weight <- sum.weight + weight[gender == "M"][n]
}
list(sum.weight, n)
